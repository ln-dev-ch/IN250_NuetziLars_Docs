# Markdown Basics

## Titles / Header

# Header 1 `# Header 1`

## Header 2 `## Header 2`

### Header 3 `### Header 3`

## Basic formatting

**bold text** `**bold text**`

*cursive* `*cursive*`

## Code-Blocks

` code one-line ` 
`` `code one-line` ``

``` 
code 
multi-line 
```
``  ```code multi-line``` ``

2. List entry 2 ordered `2. List entry 1 ordered `
3. List entry 3 ordered `3. List entry 1 ordered `

- List entry unordered `- List entry unordered `
- List entry unordered `- List entry unordered `

- [] pending task `- [] pending task`
- [x] completed task `- [x] completed task `
